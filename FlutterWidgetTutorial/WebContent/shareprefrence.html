SharedPreferences Tutorial

SharedPreference is a key-value pair relationship that is used to store data in app memory in key-value format. Means every key will have some specific value which is used for storing and retrieving the value.

We can only add int, String, double and bool using SharedPreferences.

You need to add dependency

shared_preferences:

Import the package

import 'package:shared_preferences/shared_preferences.dart';

Set value in SharedPreference : 

saveData() async {
  SharedPreferences sp = await SharedPreferences.getInstance();
  sp.setString("name", "Deep Singh");
}

Here name is a key and value as a String. We can also store different type of values like : setInt(); setDouble(); setBool();

Get value from SharedPreference : 

String getData() async {
  SharedPreferences sp = await SharedPreferences.getInstance();
  String value=sp.getString("name");
  return value;
}

Here using getString() method we can get name passing the key. We can also get different type of values like : getInt(); getDouble(); getBool();

To insure null check we can use below syntax:

String value = await getValue() ?? "";
